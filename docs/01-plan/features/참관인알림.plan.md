## 1. Overview

### 1.1 Purpose

중앙선거관리위원회 공지사항 게시판에서 "참관인" 키워드가 포함된 새로운 게시물이 게시되면 자동으로 이메일 알림을 발송하는 시스템을 구축합니다. 사용자가 매일 게시판을 수동으로 확인하는 번거로움을 없애고 중요한 정보를 놓치지 않도록 합니다.

### 1.2 Background

선거 참관인 모집 등 중요한 공지가 게시판에 올라오지만, 바쁜 일상 속에서 매일 확인하기 어렵습니다. 자동화된 알림 시스템을 통해 놓치지 않고 신청할 수 있도록 합니다.

### 1.3 Related Documents

- 타겟 게시판: https://su.nec.go.kr/su/bbs/B0000265/list.do?menuNo=200065
- Vercel Cron Jobs 문서: https://vercel.com/docs/cron-jobs
- Nodemailer 문서: https://nodemailer.com/

## 6. Architecture Considerations

### 6.1 Project Level Selection

| Level | Characteristics | Recommended For | Selected |
|-------|-----------------|-----------------|:--------:|
| **Starter** | Simple structure (`components/`, `lib/`, `types/`) | Static sites, portfolios, landing pages | ☐ |
| **Dynamic** | Feature-based modules, BaaS integration (bkend.ai) | Web apps with backend, SaaS MVPs, fullstack apps | ☑ |
| **Enterprise** | Strict layer separation, DI, microservices | High-traffic systems, complex architectures | ☐ |

**선정 사유**: 간단한 API 기능이지만 이메일 발송, 예약 작업 등 백엔드 기능이 필요하며 향후 확장성을 고려하여 Dynamic 레벨 선택

### 6.2 Key Architectural Decisions

| Decision | Options | Selected | Rationale |
|----------|---------|----------|-----------|
| Framework | Next.js / Express / Cloudflare Workers | Next.js | Vercel Cron Jobs 네이티브 지원 |
| Web Scraping | Cheerio / Puppeteer / Playwright | Cheerio | 가볍고, 정적 HTML 파싱에 충분 |
| Email | Nodemailer / SendGrid / AWS SES | Nodemailer + Gmail | 개인 프로젝트, 무료, 설정 간편 |
| State Storage | Vercel KV / JSON 파일 / DB | Vercel KV | 간단한 중복 방지, 무료 티어 충분 |
| Deployment | Vercel / Netlify / AWS Lambda | Vercel | Cron Jobs 무료 지원, 간편한 배포 |
